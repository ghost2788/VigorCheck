# CalTracker Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scaffold the CalTracker Expo + Convex monorepo with the themed navigation shell, full database schema, and base UI components — a running app with styled placeholder screens in both dark and light mode.

**Architecture:** Expo managed workflow with Expo Router for file-based tab navigation. Convex provides the backend (database, server functions, file storage). A custom theme system delivers the locked-in dark/light color palettes through React context. Base UI components (Card, ThemedText, Button) consume the theme for consistent styling.

**Tech Stack:** React Native, Expo SDK 52, Expo Router v4, Convex, TypeScript, Jest + React Native Testing Library

---

## Project Plan Roadmap

This is **Plan 1 of 6** for the CalTracker app:

| # | Plan | Delivers |
|---|------|----------|
| **1** | **Foundation (this plan)** | Running app with themed nav shell, Convex schema, base components |

- [ ] **Step 7: Implement ThemeProvider**

Create `lib/theme/ThemeProvider.tsx`:

```tsx
import React, { createContext, useContext } from "react";
import { useColorScheme } from "react-native";
import { colors, Theme, ThemeMode } from "./colors";

interface ThemeContextValue {
  theme: Theme;
  mode: ThemeMode;
}

const ThemeContext = createContext<ThemeContextValue>({
  theme: colors.dark,
  mode: "dark",
});

export function ThemeProvider({ children }: { children: React.ReactNode }) {
  const colorScheme = useColorScheme();
  const mode: ThemeMode = colorScheme === "light" ? "light" : "dark";
  const theme = colors[mode];

  return (
    <ThemeContext.Provider value={{ theme, mode }}>
      {children}
    </ThemeContext.Provider>
  );
}

export function useTheme(): ThemeContextValue {
  return useContext(ThemeContext);
}
```

import { Text, TextProps, StyleSheet } from "react-native";
import { useTheme } from "../lib/theme/ThemeProvider";

type Variant = "primary" | "secondary" | "tertiary" | "muted" | "accent1" | "accent2" | "accent3";
type Size = "xs" | "sm" | "md" | "lg" | "xl" | "hero";

interface ThemedTextProps extends TextProps {
  variant?: Variant;
  size?: Size;
  children: React.ReactNode;
}

const sizeMap: Record<Size, { fontSize: number; fontWeight: string }> = {
  xs: { fontSize: 9, fontWeight: "500" },
  sm: { fontSize: 10, fontWeight: "500" },
  md: { fontSize: 13, fontWeight: "400" },
  lg: { fontSize: 15, fontWeight: "300" },
  xl: { fontSize: 24, fontWeight: "200" },
  hero: { fontSize: 44, fontWeight: "200" },
};

export function ThemedText({
  variant = "primary",
  size = "md",
  style,
  children,
  ...props
}: ThemedTextProps) {
  const { theme } = useTheme();

  const colorMap: Record<Variant, string> = {
    primary: theme.text,
    secondary: theme.textSecondary,
    tertiary: theme.textTertiary,
    muted: theme.textMuted,
    accent1: theme.accent1,
    accent2: theme.accent2,
    accent3: theme.accent3,
  };

  const sizeStyle = sizeMap[size];

  return (
    <Text
      style={[
        { color: colorMap[variant], fontSize: sizeStyle.fontSize, fontWeight: sizeStyle.fontWeight as any },
        style,
      ]}
      {...props}
    >
      {children}
    </Text>
  );
}
```

import { Pressable, PressableProps, StyleSheet } from "react-native";
import { useTheme } from "../lib/theme/ThemeProvider";
import { ThemedText } from "./ThemedText";

interface ButtonProps {
  label: string;
  onPress: () => void;
  variant?: "primary" | "secondary";
  style?: PressableProps["style"];
}

export function Button({ label, onPress, variant = "primary", style }: ButtonProps) {
  const { theme } = useTheme();

  const isPrimary = variant === "primary";

  return (
    <Pressable
      onPress={onPress}
      style={({ pressed }) => [
        styles.button,
        {
          backgroundColor: isPrimary
            ? `rgba(${hexToRgb(theme.accent1)},${pressed ? 0.2 : 0.12})`
            : theme.card,
          borderColor: isPrimary
            ? `rgba(${hexToRgb(theme.accent1)},0.2)`
            : theme.cardBorder,
        },
        style as any,
      ]}
    >
      <ThemedText
        variant={isPrimary ? "accent1" : "primary"}
        size="md"
        style={styles.label}
      >
        {label}
      </ThemedText>
    </Pressable>
  );
}

// Convert hex to rgb values for rgba()
function hexToRgb(hex: string): string {
  const r = parseInt(hex.slice(1, 3), 16);
  const g = parseInt(hex.slice(3, 5), 16);
  const b = parseInt(hex.slice(5, 7), 16);
  return `${r},${g},${b}`;
}

const styles = StyleSheet.create({
  button: {
    borderWidth: 1,
    borderRadius: 14,
    paddingVertical: 14,
    paddingHorizontal: 20,
    alignItems: "center",
  },
  label: {
    fontWeight: "500",
    fontSize: 14,
    letterSpacing: 0.5,
  },
});
```
