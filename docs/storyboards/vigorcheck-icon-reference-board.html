VigorCheck icon references

Pick the game direction first, then we design the app heart around it.

This board is not “what the icon should be.” It is six visual lanes from actual games, so you can point at the vibe you want: premium fantasy, sci-fi medical, iconic silhouette, raw heart read, or something more graphic.

What to judge: heart readability, game feel, and whether it can evolve into a wellness icon without looking childish.

Elegant emblem

Curled heart symbol

Kingdom Hearts

Smooth, ornamental, and unmistakably “game icon” without feeling retro. This is the cleanest reference if you want a red heart that still feels premium.

  • Strong heart read at a glance
  • Can pair well with HUD brackets
  • Feels branded, not medical-logo generic
Source reference

Sci-fi medical

Optican HUD kiosk

Halo 3: ODST

This is less about the heart and more about the surrounding language: scan brackets, utility, kiosks, hard sci-fi health UX.

  • Best reference for the frame, not the icon
  • Feels like a game UI system instead of a logo
  • Could make VigorCheck feel more tactical
Source reference

Material-rich

Health globe

Diablo

A good lane if you want the app to feel powerful and dramatic. Less “heart icon,” more premium health vessel with weight and glow.

  • High drama, high polish
  • Feels expensive and game-like
  • Might be too fantasy-heavy for wellness
Source reference

Iconic silhouette

Ancient mask

Hollow Knight

Not a heart, but a great example of a health symbol with real identity. Very clean silhouette, very memorable, very little clutter.

  • Best lesson here is shape simplicity
  • Calm, iconic, and not noisy
  • Could inspire a stronger VigorCheck mark than a literal heart
Source reference

Classic game heart

Player heart

Terraria

Very clear and instantly readable, but this is the danger lane too. It goes “video game” immediately, which may be too literal for the app.

  • Best if you want a direct health-heart read
  • Useful as a shape reference only
  • Probably too retro to copy closely

Raw heart read

HUD red heart

The Binding of Isaac

Strong read, immediate, slightly gross, very visceral. Good for deciding how literal or organic you want the VigorCheck heart to feel.

  • Most literal heart in the set
  • Shows the edge of “too fleshy” fast
  • Helpful as a boundary marker, maybe not the destination
